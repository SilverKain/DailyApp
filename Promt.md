# МЕГАПРОМТ ДЛЯ AI АГЕНТА

## React приложение ежедневника (русский интерфейс)

Ты опытный Senior Frontend разработчик.
Твоя задача — создать современное приложение ежедневника (task manager / planner) на React.

Приложение должно быть:

* быстрым
* минималистичным
* полностью работающим без backend
* готовым для деплоя на GitHub Pages
* адаптивным (desktop + мобильный телефон)

---

# Технологии

Использовать:

* React
* Vite
* TypeScript
* React Hooks
* Context API
* CSS modules или Tailwind
* react-router

Приложение должно быть SPA.

---

# Хранение данных

На первом этапе использовать localStorage.

Создать абстракцию хранилища:

storageService

importFromJSON(file)

# Главная идея интерфейса

Главный экран должен выглядеть как **дашборд**.

Он разделён на несколько зон.

---

# Структура интерфейса (Desktop)

Экран делится на 5 основных зон.

Верхняя левая зона:

Проекты

Показывает список проектов.

Каждый проект можно открыть и увидеть задачи внутри.

---

Под проектами:

Сегодня

Показывает задачи на текущий день.

Функции:

* отметить выполненной
* редактировать
* удалить

---

Правая верхняя зона:

Календарь

Показывает календарь текущего месяца.

Функции:

* переключение месяца
* клик по дате показывает задачи

---

Правая нижняя зона:

Входящие

Это быстрый список задач без даты.

Здесь можно:

* быстро добавить задачу
* редактировать
* удалить

---

Центральная зона:

Детали задачи

Если пользователь кликает на задачу, здесь показывается:

название
описание
дата
проект
приоритет

Можно редактировать.

---
